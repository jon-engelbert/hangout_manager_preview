MVP
As an active participant in an active scrum ,
In order to manage (e.g. restart, reset the link, etc) the hangout,
I'd like a link from the active scrums table.

As a user,
In order to join an existing scrum,
I'd like a link from the active scrums table.

As a user,
In order to start an existing scrum,
I'd like a link to Initiate the hangout from the Upcoming scrums table

As a user,
In order to edit any scrum,
I'd like a link from the scrums tables (or possibly this is part of Managing the hangout)


NOT MVP
As a user,
In order to attend a scrum on time,
I'd like to set up a 'reminder', e.g. an event on my google calendar

As a user,
In order to see more information about an existing scrum,
I'd like a link to get more information

As a user,
In order to discuss scrums with others,
I'd like to chat with them about the scrums
